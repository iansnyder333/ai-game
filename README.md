## How to Download 
```sh
git clone https://github.com/iansnyder333/ai-game.git
cd ai-game
python3.7 -m venv venv
source venv/bin/activate
pip3.7 install -r requirements.txt
```

## How to Run 
```sh
python3.7 main.py
```



## Notes 
Python 3.7 required, any Python version >=3.7 and <3.11 will work.
